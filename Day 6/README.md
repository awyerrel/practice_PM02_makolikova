# GUI для таблицы "Фитнес-клуб"

## Запуск
1. Установите зависимости: `pip install mysql-connector-python`
2. Убедитесь, что MySQL запущен и БД существует
3. Запустите: `python app.py`

## Возможности
- Просмотр всех записей
- Добавление нового клиента
- Редактирование выбранного клиента
- Удаление клиента
- Очистка полей ввода
- Поиск по клиентам
- Обновление списка записей
## Технологии
- Python 3.x
- Tkinter (GUI)
- MySQL Connector
- mysql-connector-python
